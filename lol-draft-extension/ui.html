FIRST-TIME SETUP

1
Open League of Legends
Make sure the League client is running and you're logged in. The lockfile only exists while the client is open.
2
Paste your lockfile contents
Open Notepad, then drag the lockfile into it from:
C:\Riot Games\League of Legends\lockfile

You'll see one line like LeagueClient:1234:56789:abc123:https
Copy that whole line and paste it below.
3
Accept the League certificate
League uses a local HTTPS connection. You need to accept its self-signed certificate once.

Click the button below — a new tab will open. Click Advanced → Proceed, then come back here.
4
Done — start drafting
Queue up for a ranked game. The draft board appears automatically when you enter champion select.
